# RxUserDefaults

Rx for `UserDefaults`.

## Installation

### Swift Package Manager

Add RxUserDefaults to the `dependencies` in your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/marcbaldwin/RxUserDefaults.git", from: "3.2.0"),
]
```

Or add it in Xcode via **File → Add Package Dependencies…** using the repository URL:

```
https://github.com/marcbaldwin/RxUserDefaults.git
```

Then add `RxUserDefaults` to your target's dependencies.
